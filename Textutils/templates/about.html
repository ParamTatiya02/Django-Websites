Welcome to Textutils.in

Here is a brief info about all the string manipulation that I have used so far. Please take note that this is a work in progress and can be improved enev better. If you do so notice any correction or any suggestions, please send me an email.

Features We cover

  • Remove Punctuations – This Feature Allows you to remove all the punctuations such as ",./;'[]\/*<>?:"{}|()" from the given text .

  • New Line Remover – This Feature Allows you to remove all the extra new lines which are sometimes printed by mistake from the given user .

  • Capitalise – This Feature Allows you to capitalise all the text provided by the user .

  • Extra Space Remover – This Feature Allows you to remove all the extra spaces present between the tex .
